How To make a Adrino uno microcontroler

1. TACTILE SWITCH T842031 -160R-TR-7260

2. FERIRITE CHIP INDUCTOR 220 OHM 0305 BLM22119N1D

3. DIODE SMALL SIG 100V 0.15A 1208 CD1206_901575

Hardware

An Arduino bootloader ATmega328/168 or any compatible Atmel IC. Or a blank ATmega328, ATmega168, ATmega 8 IC, or any other compatible Atmel chips.

1.LM7805 IC.

2.16 Mhz crystal.

3.Tactile button.

4.Led’s.

5.Capacitors – 22pF (2 Nos), 10uF (2 Nos), 0.1uF (2 Nos).

6.Resistors – 220Ohm (2 Nos), 1k (2 Nos).

7.Pin headers (Male).

8.28 pin IC socket.

9.Avr pocket programmer (Optional, only if uploading bootloader).

Software

Arduino IDE

1.Terminal for Linux users and Command prompt for windows.

2.Avrdude.

Tools

1.Soldering iron.

2.Solder lead.

3.Prototyping solder dot board/breadboard or materials to etch your own board. Connecting the Components Together to Make an Arduino Board
